# 上海海港足球俱乐部数据查询应用

一个专为上海海港足球俱乐部球迷设计的数据查询应用，提供全面的球队数据查询服务，包括球队基本信息、赛季数据、历史排名、进球助攻榜和详细赛事报告等功能。

**当前赛季**: 2026赛季  
**最后更新**: 2026年4月6日  
**项目状态**: 🟢 正常运行中

## 功能板块

### 1. 球队基本信息

**一线队阵容信息**

- 展示上海海港一线队完整球员名单
- 包含球员号码、姓名、出生日期、年龄、位置等详细信息
- 支持按球员姓名搜索和按位置过滤
- 提供球员中英文名字统一翻译显示

**B队阵容信息**

- 展示上海海港B队球员阵容
- 包含球员基本信息和位置分布
- 支持搜索和筛选功能

### 2. 当前赛季数据

**一线队赛季数据**

- 展示当前赛季（2026赛季）中超联赛完整赛程
- 包含30轮比赛的对阵情况和比赛结果
- 支持按轮次、日期、对手、城市搜索和筛选
- 支持按主场/客场和比赛状态（全部/已结束/未开始）过滤

**B队赛季数据**

- 展示上海海港B队当前赛季的比赛数据
- 包含对阵情况和比赛结果
- 支持按日期和对手搜索

### 3. 历史赛季排名情况

**全赛季排名汇总**

- 展示上海海港自2006-2025年的全部赛季排名汇总
- 包含每个赛季的联赛排名、比赛场次、胜负平数据
- 展示进球数、失球数和积分等核心数据
- 支持按赛季年份搜索和排序

**联赛级别统计**

- 区分不同联赛级别（中超、中甲、中乙等）的成绩
- 展示球队在不同时期的联赛表现
- 提供赛季成绩趋势分析

### 4. 历史赛季进球助攻榜

**进球榜汇总**

- 展示2006-2025年各赛季的球员进球榜
- 包含球员姓名、进球数量、赛季年份等信息
- 支持按进球数排序和按赛季筛选

**助攻榜汇总**

- 展示2006-2025年各赛季的球员助攻榜
- 包含球员姓名、助攻数量、赛季年份等信息
- 支持按助攻数排序和按赛季筛选

**历史总榜**

- 统计球员职业生涯总进球数和总助攻数
- 展示历史最佳射手和助攻王排名
- 支持按总进球数和总助攻数排序

### 5. 历史比赛详细参考

**历史交锋情况**

- 支持查询上海海港与各支球队的历史交锋记录
- 展示两队之间的胜负平统计和进球数据
- 支持按对手球队搜索和筛选

**按赛季对阵结果**

- 展示2006-2025年各赛季的完整对阵结果
- 包含每场比赛的日期、比分、进球球员等信息
- 支持按赛季年份和比赛结果筛选

**详细赛事报告**

- 提供部分场次的详细赛事报告
- 包含比赛基本信息、球员名单、比赛事件、进球记录等
- 支持半场比分显示和进球标记（点球PK、乌龙球OG）
- 所有英文内容已统一翻译为中文
- **移动端优化**：完整的响应式设计，适配各种移动设备
- **动态半场比分**：根据进球记录自动计算半场比分

## 技术栈

- **前端**：HTML5, CSS3, JavaScript (ES6+)
- **框架**：React 17 (通过CDN引入)
- **构建工具**：无（纯静态HTML应用）
- **部署**：支持GitHub Pages, Vercel, Netlify等静态网站托管平台

## 如何运行

### 方法一：直接打开

1. 下载项目文件
2. 直接在浏览器中打开 `index.html` 文件

### 方法二：本地服务器（推荐）

1. 下载项目文件
2. 在项目目录中运行：
   ```bash
   # 使用Python 3
   python -m http.server 8000

   # 或使用Python 2
   python -m SimpleHTTPServer 8000

   # 或使用Node.js
   npx http-server -p 8000
   ```
3. 在浏览器中访问：`http://localhost:8000`

## 数据说明

### 球队数据

- 包含上海海港一线队和B队的完整球员信息
- 数据来源：上海海港足球俱乐部官方名单
- 支持中英文名字统一翻译显示

### 赛季数据

- 包含2006-2025年各赛季的完整赛程和比赛结果
- 2025赛季包含30轮中超联赛完整数据
- 数据来源：中超联赛官方赛程和比赛结果

### 历史数据

- 包含2006-2025年的联赛排名、进球助攻榜等历史数据
- 数据经过整理和验证，确保准确性

### 赛事报告数据

- 包含部分场次的详细赛事报告
- 数据格式：JSON文件，包含比赛信息、球员名单、比赛事件、进球记录等
- 所有英文内容已统一翻译为中文
