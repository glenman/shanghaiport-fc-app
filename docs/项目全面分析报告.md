```
shanghaiport-fc-app/
├── src/                          # React源代码

│   │   ├── MatchReport.tsx      # 赛事报告组件
│   │   ├── Players.tsx          # 球员信息组件
│   │   ├── Schedule.tsx         # 赛程组件
│   │   ├── Seasons.tsx          # 赛季数据组件
│   │   ├── Statistics.tsx       # 统计数据组件
│   │   └── Stats.tsx            # 数据统计组件
│   ├── App.tsx                   # 主应用组件
│   └── main.tsx                  # 应用入口
├── public/                       # 静态资源
│   ├── data/                     # 数据文件
│   │   ├── history/              # 历史比赛数据
│   │   │   ├── 2023/            # 2023赛季数据
│   │   │   ├── 2024/            # 2024赛季数据
│   │   │   └── 2025/            # 2025赛季数据
│   │   ├── 2026-*.json          # 2026赛季比赛数据
│   │   ├── schedule.json        # 当前赛季赛程
│   │   ├── schedule_b.json      # B队赛程
│   │   ├── players.json         # 一线队球员数据
│   │   └── players_b.json       # B队球员数据
│   ├── match-report.html        # 赛事报告页面(v1)
│   ├── match-report-v2.html     # 赛事报告页面(v2)
│   └── history-match-report.html # 历史赛事报告页面
├── dist/                         # 构建输出目录
├── scripts/                      # 数据处理脚本
├── datafile/                     # 原始数据文件
└── docs/                         # 项目文档
```

### 数据文件分析

| 数据类型 | 文件位置 | 更新频率 | 维护难度 |
|---------|---------|---------|---------|
| 当前赛程 | public/data/schedule.json | 每轮比赛后 | 低 |
| B队赛程 | public/data/schedule_b.json | 每轮比赛后 | 低 |
| 球员数据 | public/data/players.json | 赛季开始/转会期 | 中 |
| 比赛报告 | public/data/YYYY-MM-DD-*.json | 每场比赛后 | 高 |
| 历史数据 | public/data/history/ | 季度/年度 | 低 |
