## 🎯 核心功能模块

### 1. 球队赛程

**功能特性**:
- 显示2025赛季30轮中超联赛赛程
- 支持按轮次、日期、对手、城市搜索
- 支持筛选主场/客场比赛
- 支持按比赛状态（全部/已结束/未开始）过滤
- 已结束比赛可点击查看赛事报告
- 自动计算星期几并只显示最后一个字

**数据来源**: 2025赛季赛程数据

### 2. 球员信息

**功能特性**:
- 展示2025赛季一线队球员信息
- 支持按球员姓名搜索
- 支持按位置过滤
- 包含球员号码、年龄、国籍、身高、体重等信息
- 支持中英文名字统一翻译显示

**数据来源**: `public/data/history/2025/players.json`

### 3. 历史赛季排名

**功能特性**:
- 展示上海海港过往赛季的联赛成绩
- 包含排名、比赛场次、胜负平、进球失球和积分等数据
- 支持搜索功能（赛季、联赛）
- 2025赛季完整数据已全面汉化

### 4. 赛事报告

**功能特性**:
- 提供每场比赛的详细赛事报告
- 包含比赛基本信息、球员名单、比赛事件、进球记录等
- 支持半场比分显示和进球标记（点球PK、乌龙球OG）
- 所有英文内容已统一翻译为中文
- 动态加载对应比赛的JSON数据

**数据来源**: `public/data/history/2025/*.json`

### 5. 数据统计

**功能特性**:
- 球员数据统计：出场次数、进球、助攻、黄牌、红牌
- 球队成绩统计：主场、客场和总计的比赛数据
- 进球记录测试验证，确保数据准确性（测试成功率100%）

## 💾 数据管理

### 数据源类型

1. **JSON文件**: 主要数据存储格式，位于 `public/data/history/2025/` 目录
2. **CSV文件**: 原始数据源，位于 `datafile/` 目录
3. **翻译映射表**: 中英文名字映射，位于 `scripts/translate-2025.js`

### 数据文件说明

| 文件名 | 路径 | 用途 | 数据量 |
|-------|------|------|--------|
| players.json | public/data/history/2025/ | 球员信息 | 完整一线队球员 |
| 比赛报告JSON | public/data/history/2025/ | 详细的赛事报告 | 30场比赛 |
| 对阵结果CSV | datafile/ | 原始比赛数据 | 30场比赛 |
| 球员名单CSV | datafile/ | 原始球员数据 | 完整一线队球员 |

### 数据处理脚本

项目包含多个Node.js脚本用于数据处理：

| 脚本名 | 功能 |
|-------|------|
| translate-2025.js | 2025赛季数据汉化 |
| test-goals.js | 进球记录测试验证 |
| generate-players-json.js | 生成球员JSON数据 |

## 📝 维护指南

### 数据更新流程

1. **球员数据更新**:
   - 修改 `datafile/上海海港2025一线队大名单.csv` 文件
   - 运行 `scripts/generate-players-json.js` 生成新的players.json

2. **赛程更新**:
   - 修改 `datafile/上海海港2025中超联赛对阵结果.csv` 文件
   - 更新对应的比赛JSON文件

3. **赛事报告更新**:
   - 在 `public/data/history/2025/` 目录下创建或更新 `{date}-{type}-{round}.json` 文件
   - 确保JSON格式与现有格式一致

### 功能扩展指南

1. **添加新组件**:
   - 创建新的React组件文件
   - 在主应用中添加导航菜单和组件渲染

2. **添加新数据**:
   - 创建新的JSON数据文件
   - 添加数据加载和处理逻辑

3. **修改现有功能**:
   - 直接修改对应的组件文件
   - 确保修改后功能正常且不影响其他部分

### 常见问题解决

1. **数据加载失败**:
   - 检查JSON文件路径是否正确
   - 验证JSON文件格式是否合法
   - 检查网络连接

2. **赛事报告无法加载**:
   - 确保从赛程或历史比赛页面进入
   - 检查对应的JSON文件是否存在
   - 确保使用 `encodeURIComponent` 处理中文参数

3. **球员名字翻译问题**:
   - 在 `scripts/translate-2025.js` 中添加或修改翻译映射
   - 重新运行翻译脚本处理所有文件
