## 1. 测试环境要求

### 1.1 依赖安装


- 数据文件已放置在正确目录（`data/` 或 `public/data/`）

```bash
python test/e2e_test.py
```

## 5. 已知问题与解决方案

### 5.1 文件路径问题

**问题描述**：测试脚本无法找到数据文件

**解决方案**：在测试代码中添加路径回退逻辑
```python
file_path = PUBLIC_DATA_DIR / "filename.json"
if not file_path.exists():
    file_path = DATA_DIR / "filename.json"
```

### 5.2 搜索功能测试失败

**问题描述**：搜索一线队球员时返回无结果

**原因**：测试时可能停留在B队Tab，导致搜索一线队球员无结果

**解决方案**：在执行搜索前，先点击"一线队"Tab
```python
first_team_tab = self.page.locator(".team-tab:has-text('一线队')")
if first_team_tab.count() > 0:
    first_team_tab.click()
    time.sleep(0.5)
```

### 5.3 Header高度不一致

**问题描述**：sidebar-header与app-header高度不一致

**解决方案**：通过CSS设置固定min-height
```css
.sidebar-header {
  min-height: 94px;  /* 移动端 */
}

@media (min-width: 769px) {
  .sidebar-header {
    min-height: 130px;  /* PC端 */
  }
}
```
