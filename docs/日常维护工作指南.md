## 🔄 日常更新工作

### 每轮比赛后 (必须)

#### 1. 更新比赛结果

**时间**: 每轮比赛结束后2小时内  
**难度**: ⭐ (简单)  
**频率**: 每轮1次

**操作步骤**:

1. 打开文件: `public/data/schedule.json`
2. 找到对应轮次的比赛记录
3. 更新以下字段:
   - `result`: 比分结果 (如 "2-1")
   - `status`: 更新为 "已结束"

**示例**:
```json
{
  "id": 5,
  "round": "第5轮",
  "date": "2026-04-11",
  "time": "19:35",
  "homeTeam": "上海海港",
  "awayTeam": "山东泰山",
  "venue": "上海体育场",
  "city": "上海",
  "result": "3-1",           // 更新实际比分
  "status": "已结束"          // 更新状态
}
```

**注意事项**:
- 确保比分格式正确 (主队-客队)
- 注意主客场比赛的比分顺序

---

#### 2. 创建比赛报告 (推荐)

**时间**: 比赛结束后24小时内  
**难度**: ⭐⭐⭐ (中等)  
**频率**: 每轮1次

**操作步骤**:

1. 复制模板文件: `template/full_match_report.json`
2. 重命名为: `public/data/{日期}-{类型}-{轮次}.json`
   - 日期格式: YYYY-MM-DD
   - 类型: 中超/中乙/足协杯等
   - 轮次: 第1轮/第2轮等
   - 示例: `2026-04-11-中超-第5轮.json`

3. 填写比赛数据:

**基本信息** (`match` 部分):
```json
{
  "match": {
    "id": 5,
    "round": "第5轮",
    "date": "2026-04-11",
    "time": "19:35",
    "homeTeam": "上海海港",
    "awayTeam": "山东泰山",
    "venue": "上海体育场",
    "city": "上海",
    "result": "3-1"
  }
}
```

**比赛详情** (`matchDetails` 部分):
```json
{
  "matchDetails": {
    "temperature": "18°C",          // 比赛时温度
    "weather": "晴",                // 天气情况
    "stadiumCapacity": "37000",     // 体育场容量
    "pitchCondition": "良好"        // 场地状况
  }
}
```

**阵容信息** (`lineups` 部分):
```json
{
  "lineups": {
    "home": {
      "starting": [
        {"number": 1, "name": "颜骏凌", "position": "GK"},
        {"number": 2, "name": "李昂", "position": "DF"},
        // ... 其他首发球员
      ],
      "substitutes": [
        {"number": 12, "name": "陈威", "position": "GK"},
        // ... 其他替补球员
      ],
      "coach": "穆斯卡特"
    },
    "away": {
      // ... 客队信息
    }
  }
}
```

**比赛事件** (`events` 部分):
```json
{
  "events": [
    {
      "type": "goal",           // 事件类型
      "minute": 23,             // 时间(分钟)
      "team": "home",           // 队伍
      "player": "武磊",         // 球员
      "assist": "奥斯卡",       // 助攻(如有)
      "description": "进球描述" // 详细描述(可选)
    },
    {
      "type": "substitution",
      "minute": 65,
      "team": "home",
      "playerOut": "武磊",
      "playerIn": "吕文君"
    }
    // ... 其他事件
  ]
}
```

**比赛统计** (`statistics` 部分):
```json
{
  "statistics": {
    "home": {
      "shots": 15,
      "shotsOnTarget": 8,
      "possession": "58%",
      "corners": 6,
      "fouls": 12,
      "yellowCards": 1,
      "redCards": 0
    },
    "away": {
      // ... 客队统计
    }
  }
}
```

**注意事项**:
- 球员姓名使用中文名
- 时间使用阿拉伯数字
- 统计数据要准确
- 事件按时间顺序排列

## ⚠️ 常见问题处理

### 1. 比分显示错误

**问题**: 比赛结果显示不正确  
**原因**: `schedule.json` 中的比分格式错误  
**解决**: 
- 检查比分格式是否为 "数字-数字"
- 确认主客队顺序
- 验证JSON格式是否正确

---

### 2. 比赛报告无法加载

**问题**: 点击查看报告后页面空白  
**原因**: 
- JSON文件不存在
- 文件名格式不正确
- JSON格式错误

**解决**:
- 确认文件路径和名称正确
- 检查URL参数是否正确
- 验证JSON文件格式

---

### 3. 球员信息不显示

**问题**: 新加入球员信息不显示  
**原因**: `players.json` 未更新  
**解决**:
- 更新 `players.json` 文件
- 添加球员中文名翻译
- 检查球员号码是否重复

---

### 4. 历史数据不一致

**问题**: 历史比赛数据与实际不符  
**原因**: 数据录入错误或未及时更新  
**解决**:
- 对比官方数据源
- 更新错误数据
- 重新生成相关统计
