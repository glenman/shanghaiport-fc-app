**分析日期**: 2026年6月19日

## 📝 项目概述

**项目名称**: shanghaiport-fc-app - 上海海港足球俱乐部数据查询应用

**项目定位**: 专为上海海港足球俱乐部球迷设计的数据查询应用，提供球队赛程、球员信息、历史赛季成绩、赛事报告和数据统计等功能。支持一线队（中超联赛、足协杯、亚冠）和B队（中乙联赛）的数据管理。

**创建时间**: 2025年

**最后更新**: 2026年6月

## 📈 近期更新日志 (2026年5月-6月)

### 新增功能

1. **比赛类型数据管理**
   - 创建 `competitions.json` 比赛类型映射文件
   - 定义比赛类型全称、简称和类型标识（CSL/CFA/ACLE/C2L）
   - 为B队所有赛事报告添加 `competition` 字段

2. **按比赛类型分组统计**
   - 当季数据统计页面支持按比赛类型切换显示
   - 一线队：中超(CSL)、足协杯(CFA)、亚冠(ACLE)
   - B队：中乙(C2L)
   - 进球榜、助攻榜、红黄牌分别按比赛类型统计

3. **赛事统计增强**
   - 详细数据分析页面(match-overview-stats.html)
   - 支持8个统计维度：控球与预期进球、射门、进攻、传球、对抗、防守、门将、定位球
   - 新增界外球、进攻三区进入次数等统计
   - 新增xGOT（射正预期进球）数据展示

4. **数据文件结构优化**
   - 分离一线队和B队数据目录（team-a, team-b）
   - 新增详细统计数据文件格式：`YYYY-MM-DD-*-MO.json`
   - B队赛事报告添加competition字段标识比赛类型

### 功能改进

1. **当季数据统计页面(CurrentStats.tsx)**
   - 球队概览数据根据比赛类型动态更新

2. **统计脚本优化(update_stats.py)**

4. **数据质量修复**

   - 修正球员名称（吕永涛→卢永涛）

## 📊 数据管理

### 比赛类型映射 (competitions.json)

```json
{
  "competitions": {
    "中国足球协会超级联赛": {
      "shortName": "中超",
      "type": "CSL",
      "teams": ["first-team"]
    },
    "中国足球协会杯": {
      "shortName": "杯赛",
      "type": "CFA",
      "teams": ["first-team"]
    },
    "亚足联冠军精英联赛": {
      "shortName": "亚冠",
      "type": "ACLE",
      "teams": ["first-team"]
    },
    "中国足球协会乙级联赛": {
      "shortName": "中乙",
      "type": "C2L",
      "teams": ["team-b"]
    }
  }
}
```

### 数据文件说明

| 文件 | 说明 | 更新频率 |
|------|------|---------|
| schedule.json | 一线队赛程 | 每轮比赛后 |
| schedule_b.json | B队赛程 | 每轮比赛后 |
| players.json | 一线队球员名单 | 赛季开始/转会期 |
| players_b.json | B队球员名单 | 赛季开始/转会期 |
| current_stats.json | 当季数据统计 | 每轮比赛后 |
| competitions.json | 比赛类型映射 | 赛季更新 |
| team-a/*.json | 一线队赛事报告 | 每场比赛后 |
| team-b/*.json | B队赛事报告 | 每场比赛后 |
| history/ | 历史赛季数据 | 季度/年度 |

### 统计数据字段说明

**current_stats.json 结构**:
```json
{
  "first-team": {
    "competitions": {
      "中国足球协会超级联赛": {
        "matchesPlayed": 15,
        "record": "4胜5平6负",
        "goalsFor": 25,
        "goalsAgainst": 22,
        "points": 17,
        "topScorers": [...],
        "topAssists": [...],
        "yellowCards": [...],
        "redCards": [...]
      },
      "中国足球协会杯": {...}
    }
  },
  "team-b": {
    "competitions": {
      "中国足球协会乙级联赛": {...}
    }
  }
}
```

**match_overview 统计维度**:
- `possession`: 控球率
- `expected_goals`: 预期进球(xG)
- `xgot`: 射正预期进球(xGOT)
- `shots`: 射门统计（总射门、射正、射偏、被封堵）
- `attack`: 进攻统计（进攻三区进入次数、危险进攻、界外球）
- `passing`: 传球统计（传球总数、成功率）
- `duels`: 对抗统计（总对抗、成功率）
- `defense`: 防守统计（拦截、解围、铲抢）
- `goalkeeper`: 门将统计（扑救、出击、高空球）
- `set_pieces`: 定位球统计（角球、任意球、红黄牌）

### 数据处理脚本

| 脚本名 | 功能 |
|-------|------|
| update_stats.py | 统计数据更新，支持按比赛类型分组统计 |
| normalize_match_report.py | 球员姓名标准化 |
| update_schedule.py | 赛程更新 |
